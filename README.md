# LahanShop

LahanShop — це повноцінний інтернет-магазин, розроблений з використанням сучасних технологій. Проект складається з клієнтської частини (Frontend) та серверної частини (Backend). Основна мета проекту — надання зручного інтерфейсу для перегляду та покупки товарів (наприклад, автозапчастин чи інших категорій), а також інструментів адміністрування для управління каталогом.

## Технології (Tech Stack)

- **Frontend:** React, TypeScript, React Router, TailwindCSS/CSS
- **Backend:** C#, .NET
